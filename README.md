# Density Based Smart Traffic Signal System

## Project Overview

This project is a Density Based Smart Traffic Signal System developed using Arduino in Tinkercad.

The system controls traffic lights automatically based on vehicle density instead of fixed timing, reducing waiting time and improving traffic flow.

---

## Features

- Automatic traffic signal control
- Vehicle density detection
- Dynamic signal timing
- Arduino-based implementation
- Tinkercad simulation

---

## Components Used

- Arduino Uno
- LEDs (Red, Yellow, Green)
- Ultrasonic Sensor (Simulation)
- Breadboard
- Jumper Wires
- Resistors

---

## Working

1. Sensors detect vehicle density.
2. Arduino reads sensor values.
3. Lane with higher density gets higher green time.
4. Remaining lanes wait until completion.
5. Process repeats continuously.
